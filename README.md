# Môn SWE - Lab 2 – Design Patterns

## Mục tiêu

Thực hành 8 mẫu thiết kế phần mềm phổ biến bằng TypeScript.

## Danh sách các mẫu đã thực hành

- Singleton
- Factory Method
- Builder
- Adapter
- State
- Strategy
- Observer
- Command

## Cách chạy

Vào thư mục tương ứng và chạy:

```bash
ts-node main.ts > output.txt
```
